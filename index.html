Levee
Levee is a language-agnostic rate limiting service built in Java and Spring Boot, backed by Redis. It protects external APIs and LLM integrations from quota exhaustion and cost spikes by enforcing configurable per-app and per-entity limits using pluggable algorithms — Fixed Window for API request counting and Token Bucket for variable LLM token budget management.

Chisel
A shared React component library built to maintain design consistency across a personal suite of apps. Ships six components — Button, TextBox, Chip, ChipInput, ThemeToggle and Loader — all with built-in dark/light theme support via CSS variables. Components are developed and previewed in Storybook, bundled with Rollup into ESM and CJS formats, and published to GitHub Packages. New components are scaffolded using a Plop generator that enforces a consistent file structure.
